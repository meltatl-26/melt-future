## Naming Convention

```
F-[number]-[kebab-slug].md
```

## PRD Status Definitions

- **Not Started** — no code written
- **In Progress** — code written but not user-verified
- **Pending Review** — code complete, awaiting user browser confirmation
- **Complete** — user has confirmed in browser, score 10/10
